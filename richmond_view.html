    • Main : PC(1920px), Tablet(768px), Mobile(430px)
  • 작업 기간
    • 2일
  • 벤치 마케팅
    • 한화 오션, 비에이 컴퍼니, 한화솔루션 인사이트, 트라움그룹을 보면서 RICH MOND랑 다른점이 무엇인지 비교하게 되었고 사이트에 컬러가 초록색 갈색의 따뜻한 느낌을 줘서 사용자가 보기 편함을 줍니다. 단점은 기능에 있어 많은 것이 없다는 것입니다.
  • THE RICH MOND 선택 이유
    • 건축과 자동차에 관심이 있어 사이트를 한개 하면 좋겠다고 생각하였습니다. 여러가지 작업을 배우던중 브랜드의 우아한 디자인이 인상적이었습니다. 무엇보다도, 공간의 여백과 함께 조화를 이루는 미니멀한 접근이 현대적인 감각을 잘 표현했습니다.
  • 작업 내용
    • 공통
      • 전체 페이지를 #warp안에 포함 시켰습니다.
    • Header
      • Jquery를 사용하여 마우스가 hover 되거나 focusin이 되면 슬라이드 다운 되도록 하였습니다.
    • nav
      • nav를 사용하여 안에 gnb를 만들었습니다.
    • gnb
      • 마우스가 올라가면 gnb안에 ul li이 내려오게 만들었습니다.
    • icon
      • 폰트 어썸이라는 사이트에서 아이콘을 가져왔으며 link를 연결하여 사용했습니다.
    • section
      • section에 id 값을 줘서 구역 별로 나눠서 main00을 만들었습니다.
    • footer
      • footer에는 ⓒ를 대신해서 ©라는 사용하였습니다.
    • 작업 후기
      • 생각 했던것 보다 완성후 미니멀한 느낌을 더 받았습니다. 그 이유는 기능적인 부분에서 부족하다고 생각했기 때문입니다. main부분만 만들어서 그렇지만 THE RICH MOND가 의도한 바가 맞으면 잘 표현했다고 생각합니다.
