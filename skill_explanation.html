MY SKILL 돌아가기 SKILL

  • "HyperText Markup Language" 마크업 랭귀지를 준수하여 홈페이지를 구조적으로 작성하는 방법을 배웠으며 콘텐츠를 구분하는 시멘틱 태그를 사용하여 웹 페이지에 가독성을 높이도록 하였습니다.

  • "Cascading Style Sheets"를 사용하면서 반응형 디자인에 대한 고려할 점이 많은것을 배웠습니다. 주로 flex와 transform을 통해 페이지 레이아웃을 유연하게 유지하면서도 인터랙티브한 요소를 추가해 시각적 흥미를 주며, 사용자의 경험을 향샹 시키도록 했습니다.

  • "Javascript"는 웹 페이지에 동적인 기능을 모두 담당하는것을 배웠습니다. function과 함수들을 사용하여 제가 구현 할 수 있는 부분을 구현하도록 노력 했습니다.

  • "jQuery"는 배우면서 느낀 점은 자바스크립트보다 조금 쉬운 단계라고 생각했습니다. jQuery 사용할 때 slideToggle,ToggleClass, mouseover, mouseleave, foucuson, foucusout를 사용하여 gnb를 만들었습니다.

  • "React"는 사용하지 않았으나 배우면서 느낀점은 각각 라이브러리를 나눠서 사용하였고 tab메뉴 기능을 자바스크립트 없이 구현하는걸 보고 효율적에 강점이란걸 느꼈습니다.

  • "Photoshop"은 주로 사진 보정 편집 상세페이지 디자인 작업이나 누끼를 따서 가져올때 사용하여 작업했습니다.

  • 로고나 캐릭터 만들때 펜툴을 사용하여 작업을 했습니다.

  • "Premiere_Pro"컷 편집과 자막 보이고 효과 간단한 기능 사용 가능합니다.

  • "Figma"는 디자인 레이아웃과 이미지 공간 크기를 잡을때 사용하였습니다. 레이아웃 구조를 짜면서 클릭시 이동하는 구조도 생각하게 되었습니다.

  • "Git hub"는 완성한 작업물들을 github에 올릴때 사용하였고 VsCode에 Terminal에 git bash를 사용하여 업로드 하였습니다.
