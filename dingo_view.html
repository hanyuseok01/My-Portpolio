  • Renewal 전 dingo 특징
    • 이전 사이트는 직관적이면서 강렬한 영상 콘텐츠를 메인 화면에 나오게 하였고 컨텐츠가 적은 만큼 영상 컨텐츠 스트리밍 속도를 향샹 시켰습니다. 간편한 애니메이션과 JS로 구현되어서 사용자가 사이트 내에서 자연스럽게 탐색할 수 있도록 만들었습니다.
  • Renewal 후 dingo 특징
    • "Dingo" 사이트를 만들면서 의도한 점은 동영상 콘텐츠에 중점을 두고 있으며, 주로 음악과 엔터테인먼트 관련 콘텐츠를 중점으로 만들었습니다. 심플한 레이아웃과 PC, Tablet, Moblie을 만들어 다양한 기기에서 접근하기 용이하게 설계했습니다. 특히 모든 부분은 아티스트가 만들 앨범, 뮤직, 영상, 공연, 콜라보레이션을 한 영상을 중심으로 만들었으며, 역동적이면서 비주얼 요소와 고화질 영상이 사용하였습니다.
  • 작업 내용
    • 공통
      • 모든 페이지 오른쪽에 follow 메뉴가 있다.
    • Header
      • Jquery를 사용하여 마우스가 hover 되거나 focusin이 되면 슬라이드 다운 되도록 하였습니다.
    • Section Area
      • Swiper 요소를 사용하여 음악, 콜라보레이션, 영상 컨텐츠가 이동하도록 하였습니다.
      • @keyframes를 사용하여 Transform에서 회전, 왕복 하게 하였습니다.
      • 영상 이미지 클릭시 음악을 들을수 있는 유튜브 사이트로 이동
      • 클릭시 나오고 들어오게 하는 Jquery를 사용하여 사이드 메뉴바를 만들었습니다.
      • Tab 메뉴를 사용하여 같은 메뉴에 있는 사이트로 바로 전환 가능 gnb 클릭시 페이지로 이동도 가능
    • Footer
      • 원 사이트 느낌을 훼손하지 않기 위해서 심플하게 만들었습니다.
    • 작업 후기
      • 막상 학원에서 했던 메인 페이지 한개 했던걸로 금방하니까 나도 금방 하겠지 라는 생각으로 만들었다가 절대 아니라는걸 느꼈습니다.
      • 내 실력으로 구현 불가능한것은 과감하게 보내주고 내가 구현 가능한 기능을 최대한 열심히 집중해 보았습니다. JS뿐만 아니라 CSS로 가능한 3D 입체적인 모습을 경험하였습니다.
      • 처음 욕심으로 손을 안대고 시간만 보내다가 정신 차리고 하니까 페이지가 한개씩 완성되는 모습에 내 자신이 대단했다. 시간이 생기면 풀 페이지 완성할 것입니다.
